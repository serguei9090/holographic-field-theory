## Abstract
This paper presents the experimental validation of the **Complex Holographic Field Theory (CHFT) v2** paradigm, a non-LLM language modeling framework. CHFT v2 replaces traditional multi-head self-attention and deep neural network parameters with **Fourier Holographic Reduced Representations (FHRR)** for context superposition, combined with a **Modern Hopfield Network** for attractor-based token retrieval. We address the critical bottleneck of softmax saturation in high dimensions ($D=4096$) by introducing **$\sqrt{D}$ scaling normalization** and **repetition penalization**. Furthermore, we resolve context order symmetry (bag-of-words limitation) by implementing a **Positional Phase-Shift Binding** operator. 

Experimental results on the TinyStories dataset show a **Perplexity reduction from 97,451,090.90 to 236.87** and a boost in **Accuracy@1 to 8.29%** (outperforming the frequency baseline by **+1.38pp**). We demonstrate that coherent syntactic associations and vocabulary diversity (Diversity Score of **34.0%**) can be learned without deep backpropagation or quadratic attention matrices.

## 1. Introduction & Motivation
State-of-the-art Large Language Models (LLMs) rely heavily on the Transformer architecture. While powerful, Transformers incur a quadratic computational complexity $O(N^2)$ and linear memory footprint $O(N)$ during inference due to the Self-Attention mechanism and the requirement of storing the Key-Value (KV) cache. This limits the scalability of LLMs in resource-constrained environments.

**CHFT v2** explores an alternative path: utilizing Vector Symbolic Architectures (VSA) to compress context histories into a single, fixed-size holographic state $\Psi \in \mathbb{C}^D$, resulting in a constant $O(1)$ memory complexity. Token retrieval is achieved using Modern Hopfield Memories which perform associative memory recovery in a single computation step, equivalent to parameter-free cross-attention.

## 2. Mathematical Framework

### 2.1. FHRR Phasor Embedding
Each token $t$ in the vocabulary $V$ is mapped to a static complex phasor vector $v_t \in \mathbb{C}^D$ located on the unit circle:
$$v_t = e^{i\theta_t} = \cos(\theta_t) + i\sin(\theta_t)$$
where $\theta_t \in [-\pi, \pi]^D$ is the phase parameter vector adjusted during training.

### 2.2. Positional Phase-Shift Binding
To preserve sequential order without neural parameters, we define a static phase rotation operator $P_j$ for each context index $j \in \{0, \dots, L-1\}$:
$$P_j = e^{i \cdot j \cdot \omega} = \cos\left(j \cdot \frac{\pi}{L}\right) + i\sin\left(j \cdot \frac{\pi}{L}\right)$$
For a context sequence of tokens $(c_0, c_1, \dots, c_{L-1})$, we bind each token phasor to its position and perform holographic superposition (bundling) to construct the query state $\Psi$:
$$\Psi = \sum_{j=0}^{L-1} v_{c_j} \odot P_j$$

### 2.3. Normalized Real-Arithmetic Cosine Similarity
In high dimensions ($D=4096$), the dot product of the normalized state $\Psi$ (L2 norm $= 1$) and codebook keys (L2 norm $= \sqrt{D}$) scales to $\pm\sqrt{D} = \pm 64.0$. To prevent softmax gradient saturation during cross-entropy loss calculation, we normalize the logits by $\sqrt{D}$.

To maximize GPU utilization (Nvidia Tensor Cores), the complex matrix multiplication is decomposed into equivalent real-valued operations:
$$\text{Re}(\Psi \cdot \text{conj}(K)^T) = \frac{\text{matmul}(\Psi_r, K_r^T) + \text{matmul}(\Psi_i, K_i^T)}{\sqrt{D}}$$
The final training loss is computed with a Hopfield scale factor (inverse temperature) $\beta = 16.0$:
$$\mathcal{L} = \text{CrossEntropy}(\text{Logits} \times 16.0, \text{Target})$$

## 3. Experimental Setup
* **Dataset:** TinyStories (1,000 story subset tokenized via `tiktoken` to 5,424 unique vocabulary tokens).
* **Hardware:** Google Colab GPU T4 (16GB VRAM).
* **Hyperparameters:** 
  * Dimension ($D$): 4096
  * Context Window ($L$): 8 tokens
  * Optimizer: Adam ($\text{lr} = 0.01$) with Cosine Annealing scheduler
  * Batch Size: 1024
  * Epochs: 10
  * Top-K Generation ($K$): 5
  * Temperature: 0.8
  * Repetition Penalty: $-35.0$ applied to context token logits

  > *Prompt:* 'Once upon a time'  
  > *Output:* Once upon a time time time time time time time time time time time time... (Infinite repetition due to auto-associative dominance).
* **After Positional Binding & Normalization (CHFT v2):**
  > *Prompt:* 'Once upon a time'

The output text transitions from simple word repetition to valid syntactic constructs (noun-object association: `"saw something"`, pronoun-verb-adjective: `"They were scared"`, infinitives: `"wanted ... to go"`).

## 5. Discussion & Future Work
The transition from CHFT v1 to v2 has validated the core paradigm of combining FHRR with Modern Hopfield Networks. By solving the mathematical scaling bottleneck, we proved that gradient-descent phase optimization is a viable training method for holographic systems. 

**Future directions include:**
1. **Deep Hopfield Layers:** Iterative state refinement to emulate multi-layer transformer reasoning.
2. **Infinite Context Decay:** Replacing sliding windows with a continuous temporal decay factor $\alpha \in [0.8, 0.99]$ to achieve true infinite context with $O(1)$ memory.
3. **Hyperbolic Poincaré Embeddings:** Interfacing the VSA binding space with non-Euclidean manifolds to capture taxonomic word hierarchies.
